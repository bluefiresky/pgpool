    PGPool

    PGPool 是 PostgresSQL 数据的客户端，采用了连接池设计，在出错时，自动重新连接数 据库。PGPool 并非完全从头开发，在底层利用 poolboy 来做连接池，用 epgsql 做 数据库的驱动。
